FAQs

  • How to calculate course grade:

    • Go to the grades page of your course grade
    • This should be under the url format:
      [school-name].instructure.com/courses/[courseID]/grades
    • If you want your calculated grade to be stored for grade calculation, make sure the grading period is is "All Grading Periods" (You can select this from the dropdown)
    • Alternatively, you can go to the following url format:
      [school-name].instructure.com/courses/[courseID]/grades?grading_period_id=0

  • How to view calculated GPA:

    • Go to the dashboard page of Canvas. This should follow the url format:
      [school-name].instructure.com

  • How to use auto-update:

    • Go to the Canvas dashboard page, make sure the page loads in "Card View"
    • Reload the page if your default view is not card view.
