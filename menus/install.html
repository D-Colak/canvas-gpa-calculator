Thank you for downloading Canvas GPA Calculator!

Let's get you started with some basic settings:

1. Click on the extension icon to open the menu (you should pin the extension for easier access)

2. Modify the settings to you liking (Auto Update Button currently does not work)

3. To use manual course grade input, navigate to your dashboard page on Canvas and reload the page

To calculate course grades:

1. Go to the "grades" page of a course

2. To let the calculator store your course grade fo GPA calculation, make sure your grading period is set to "All Grading Periods". This can also be done by adding the query "?grading_period_id=0" to the end of your current URL.

Bugs/errors can be reported on the Chrome Web Store, or on Github
